# 📚 Online Book Corner

An Online Book Corner web application developed using Spring Boot and Thymeleaf.

## 🚀 Features

### 👤 User
- Register & Login
- Search Books
- Buy Books
- View Orders

### 👨‍💼 Admin
- Admin Dashboard
- Add / Delete / Update Books
- View Users
- View Orders

## 🛠️ Tech Stack
- Java
- Spring Boot
- Thymeleaf
- H2 Database

## ▶️ How to Run
1. Clone the repository
2. Open in Eclipse
3. Run `Main.java`
4. Open browser at `http://localhost:8080`
